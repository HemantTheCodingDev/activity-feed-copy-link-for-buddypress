This plugin adds a "Copy Link" button to each activity feed in BuddyPress.

== Description ==

Activity Feed Copy Link For BuddyPress adds a "Copy Link" button to each activity feed in BuddyPress. This allows users to easily copy the link to a specific activity and share it with others.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/activity-feed-copy-link-for-buddypress` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.

== Frequently Asked Questions ==

= Does this plugin require BuddyPress to be installed? =

Yes, this plugin requires BuddyPress to be installed and activated on your WordPress site.
